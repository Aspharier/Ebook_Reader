<h1 align="center">Ebook Reader</h1>

<p align="center">
  A modern and minimal Android eBook reading application built with Kotlin & Jetpack Compose
</p>

---

**Ebook Reader** is a modern and minimal **Android eBook reading application** built using **Kotlin** and **Jetpack Compose**.  
It allows users to **import, read, and manage PDF books** with a smooth and distraction-free reading experience

Think of it as your **personal digital library** — simple, fast, and elegant.

## Features

- Import PDF books from local storage  
- Read PDFs using Android’s native PDF renderer  
- Clean and distraction-free reading UI  
- Library screen to manage all added books  
- Book details screen with metadata  
- Smooth navigation using **Jetpack Compose**
- Modern **Material 3** design
- Lightweight and fast

---

## Tech Stack

- **Kotlin**
- **Jetpack Compose**
- **Material 3**
- **Hilt** (Dependency Injection)
- **Coroutines**
- Android **PdfRenderer**
- **MVVM Architecture**
